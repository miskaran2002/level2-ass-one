TypeScript Assignment – Problem Solving & Blog Writing
 Introduction

This project is a TypeScript-based assignment that focuses on strengthening fundamental programming concepts such as data types, functions, interfaces, generics, and object-oriented programming. It also includes blog writing tasks to improve technical communication skills.

The main goal of this assignment is to practice writing clean, readable, and type-safe TypeScript code while following best coding practices.

 Problem Solving (50 Marks)

All coding solutions are implemented in a single file:

solutions.ts

This section includes 7 programming problems covering:

Array manipulation
String operations
Type guards and union types
Generics
Interfaces
Class inheritance
Data structure operations

Each solution is written using strict TypeScript typing to ensure correctness and maintainability.

 Key Concepts Covered
 1. Data Types & Type Safety

Using TypeScript’s strong typing system to avoid runtime errors.

 2. Functions

Writing reusable and modular functions for different operations.

 3. Interfaces

Defining object structure clearly using interface.

 4. Generics

Creating reusable functions that work with multiple data types.

 5. OOP (Object-Oriented Programming)

Using classes, inheritance, and methods to model real-world entities.

 6. Type Guards

Safely checking data types at runtime.
